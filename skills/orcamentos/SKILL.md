# Skill: Gerador de Orçamentos DAMA

## Quando usar
Use esta skill quando o usuário pedir para criar, sugerir ou montar um orçamento.

## Instruções

1. **Pesquise preços atuais** no mercado brasileiro para o tipo de serviço solicitado (agências digitais, produtoras, estúdios) antes de sugerir valores.

2. **Responda com essas informações organizadas:**

### Campo 1: Informações do Projeto
- Título do Projeto:
- Breve descrição:
- Divisão: (Tech / Sports / Studio)

### Campo 2: Serviços / Itens
Para cada serviço:
- Nome:
- Descrição:
- O que está incluso: (lista com 3-5 itens)
- Valor sugerido: R$ X (baseado na pesquisa de mercado)
- Valor mensal (se aplicável): R$ X/mês

### Campo 3: Valores
- Subtotal: R$
- Desconto sugerido (se aplicável): R$
- Total: R$
- Recorrente mensal (se houver): R$/mês

- Prazo de entrega sugerido:
- Condições de pagamento sugeridas:
- Validade do orçamento: 15 dias
